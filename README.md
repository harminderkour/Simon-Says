# Simon Says

## Overview

Simon Says is a classic memory-based game developed using **HTML**, **CSS**, and **JavaScript**. The game challenges players to memorize and repeat an increasingly complex sequence of colors. With each successful round, a new color is added to the sequence, making the game progressively more difficult and engaging.

## Features

- Interactive and responsive user interface
- Randomly generated color sequences
- Increasing difficulty with each level
- Real-time user feedback
- Score and level tracking
- Game over notification with restart functionality

## Technologies Used

- HTML5
- CSS3
- JavaScript

## Workflow

1. The player starts the game by pressing a key or clicking the start button.
2. The game generates a random color and displays it as part of the sequence.
3. The player repeats the displayed sequence by clicking the colored buttons.
4. The player's input is validated against the generated sequence.
5. If the sequence is correct, the player advances to the next level.
6. A new color is added to the sequence, increasing the game's difficulty.
7. If the player enters an incorrect sequence, the game ends and the final score is displayed.
8. The player can restart the game and try again.
